## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Customization

### Colors

The aqua color theme is defined in `tailwind.config.js`. You can customize the color palette by modifying the `colors.aqua` section.

### Content

Update the following files to customize your portfolio:
- `src/components/Hero.tsx` - Your name and introduction
- `src/components/About.tsx` - About section content and stats
- `src/components/Skills.tsx` - Your skills and technologies
- `src/components/Projects.tsx` - Your project showcase
- `src/components/Contact.tsx` - Contact information
